# 🐼 粉熊防詐檢測器

演唱會／活動票券二手交易詐騙風險評估工具

## 功能說明

此工具透過 **10 道檢測題目**，從三大維度評估二手票券交易的詐騙風險：

| 維度 | 檢測內容 |
|------|---------|
| 👤 賣家帳號可信度 | 帳號年資、貼文內容、共同好友 |
| 🚩 交易行為異常點 | 面交意願、急迫感、價格合理性、付款方式 |
| 🎫 票券資訊驗證 | 座位資訊、購票證明、取票方式 |

### 風險等級

- ✅ **低風險**（≤ 20%）— 交易相對安全
- ⚠️ **中風險**（21%～50%）— 請提高警覺
- 🚨 **高風險**（> 50%）— 極可能是詐騙
